# Balnc Nova UI

一个现代化的 Vue 3 组件库，提供丰富的 UI 组件和优秀的开发体验。

## 特性

- 🚀 基于 Vue 3 + TypeScript 开发
- 🎨 完整的主题系统
- 📦 开箱即用的高质量组件
- 🌍 支持国际化
- 📱 响应式设计
- 🔧 完善的类型定义

## 快速开始

### 安装

```bash
npm install @balnc-nova-ui/core
```

### 使用

```vue
<template>
  <BnButton type="primary">点击我</BnButton>
</template>

<script setup>
import { BnButton } from '@balnc-nova-ui/core';
</script>
```
