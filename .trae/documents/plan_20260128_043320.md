# bn-trigger 组件实现计划

## 组件概述
创建一个通用的触发器组件，用于下拉选项、菜单项、标签等可交互控件。

## 文件结构
```
bn-trigger/
├── index.ts                          # 统一导出
├── bn-trigger.tsx                    # 主组件文件
├── props/
│   └── bn-trigger-props.ts           # Props 类型定义
├── composables/
│   ├── use-trigger-classes.ts       # 样式类名计算
│   └── use-trigger-event.ts         # 事件处理
└── styles/
    ├── index.scss                   # 样式入口
    ├── variables.scss               # 组件变量
    ├── mixin-base-style.scss        # 基础样式
    ├── mixin-size-style.scss        # 尺寸样式
    ├── mixin-color-style.scss       # 颜色变量定义
    ├── mixin-variant-style.scss     # 变体样式（soft/solid）
    ├── mixin-state-style.scss       # 状态样式（hover/active/selected/disabled）
    ├── mixin-shape-style.scss       # 形状样式（circle/square/capsule）
    └── mixin-border-style.scss      # 边框样式
```

## 核心功能实现

### 1. Props 配置
- **color**: 'primary' | 'danger' | 'warning' | 'success' | 'info' | 'nature' | 'white' | 'black'
- **variant**: 'soft' | 'solid'
- **size**: 'small' | 'medium' | 'large'
- **shape**: 'circle' | 'square' | 'capsule'
- **bordered**: boolean（是否显示边框）
- **disabled**: boolean
- **selected**: boolean
- **prefixIcon**: string | VNode（前缀图标）
- **suffixIcon**: string | VNode（后缀图标）
- **onClick**: 事件回调

### 2. 插槽支持
- **prefix**: 前缀内容插槽（可覆盖 prefixIcon）
- **suffix**: 后缀内容插槽（可覆盖 suffixIcon）
- **default**: 默认内容插槽

### 3. 样式系统设计
- **Soft 变体**：使用半透明背景色（--bn-bg-color-*-soft-base），文字使用对应主色
- **Solid 变体**：使用实心背景色（--bn-bg-color-*-base），文字使用白色
- **边框处理**：bordered=false 时使用 transparent 替代边框颜色
- **状态样式**：完整支持 hover、focus、active、selected、disabled 状态
- **图标间距**：定义图标与文字之间的间距变量

### 4. 颜色变量定义
在 `mixin-color-style.scss` 中定义完整的颜色变量系统：
- 为 8 种色系（primary/danger/warning/success/info/nature/white/black）定义 soft 和 solid 变体的背景色、文字色、边框色
- 为每种状态（hover/active/selected/disabled）定义对应的颜色变量

### 5. TypeScript + TSX 实现
- 遵循 vue-code-skill 规范
- 使用 defineComponent 定义组件
- Props 回调模式处理 onClick 事件
- 支持插槽渲染（prefix、suffix、default）
- 逻辑下沉至 composables
- Ref 变量带 Ref 后缀

## 实现步骤
1. 创建目录结构和文件
2. 实现 Props 类型定义（包含图标和插槽相关类型）
3. 实现 composables（样式类名计算、事件处理）
4. 实现样式系统（variables + mixins，包含图标间距变量）
5. 实现主组件（TSX，支持插槽渲染）
6. 创建 index.ts 导出
