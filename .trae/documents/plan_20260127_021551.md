## 为 BnSelect 添加独立校验能力

### 1. 更新 select-props.ts
- 添加 `rules` 属性（类型：`FormItemRule | FormItemRule[]`）
- 添加 `error` 和 `errorMessage` 属性（如果还没有）

### 2. 更新 bn-select.tsx
- 移除 FormItem 侵入式校验（删除 `inject(FormItemContextKey)` 和相关调用）
- 引入 `useFieldValidation` composable
- 使用 `toRef` 包装 `modelValue` 和 `rules`
- 在 blur 事件中调用 `validate('blur')`
- 在 change 事件中调用 `validate('change')`（通过 watch 监听）
- 通过 `expose` 暴露校验方法：`validate`, `clearValidate`, `showError`
- 添加错误信息显示逻辑（类似 Input 组件）
- 添加错误状态样式类

### 3. 更新样式（如需要）
- 添加错误状态样式（参考 bn-input 的错误样式）

### 核心改进点
- ✅ 避免 FormItem 侵入，实现组件独立校验
- ✅ 提供与 Input 相同的校验 API
- ✅ FormItem 会自动检测并使用组件的校验能力
- ✅ 保持向后兼容（FormContext 仍然可以注入）
