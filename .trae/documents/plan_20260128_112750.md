# 项目启动和清理计划

## 1. 安装依赖
- 运行 `pnpm install` 安装所有依赖项（包括 workspace 依赖）

## 2. 启动项目
- 运行 `pnpm dev` 启动所有应用（proxy-monitor 和 balnc-nova-ui-docs）
- 或者单独运行 `pnpm dev:proxy-monitor` 只启动 proxy-monitor 应用

## 3. 删除旧 src 目录
- 删除 `h:\lingann\proxy-monitor\src` 目录（已迁移到 apps/proxy-monitor/src）

## 验证步骤
- 确认依赖安装成功
- 确认应用正常启动
- 确认旧 src 目录已删除
