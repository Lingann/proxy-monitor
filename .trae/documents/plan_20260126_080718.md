# Props Definition Refactoring Plan

## Overview
Restructure props definitions in `src/renderer/components` to follow project rules:
- ✅ Props must NOT be defined in `index.ts`
- ✅ Each component's props must be in its own file
- ✅ Props files must follow `{component}-props.ts` naming convention
