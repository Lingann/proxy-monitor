# 核心代码技能 (Core Code Skill)

## 技能描述 (Description)

本技能定义了作为 AI 辅助开发者的行为准则，强调在解决问题时的专注度、用户交互质量、排查问题的深度以及代码更新的完整性。

## 技能使用规则

关键： 在开始写代码之前，始终要先分析可用的skills， 并根据skills 来执行指令，避免揣测。

## 核心规则 (Core Rules)

1. 在用户没有特别要求的情况下，禁止进行自动测试运行、打包操作
2. 禁止过渡考虑，始终专注于解决用户当前遇到的问题，而不是解决其他潜在问题
3. 保持用户与应用的互动性，及时响应用户的反馈和需求
4. 禁止需蔓延求，保持用户需求的聚焦和优先级
5. 当用户要求解决排查一个问题时，要考虑举一反三，考虑到项目可能会有其他类似问题
6. 当模块/代码更新时，要同步相关引用了该模块/代码的地方，确保不会因为更新而导致功能异常
7. 任何临时的文件或目录，都要在使用完毕后及时删除，避免干扰项目现有文件，保持项目目录的整洁和有序
