# 更新规范 (Update Rules)

本文件定义了代码更新时的联动规范，防止引入回归错误。

## 规则列表

### 1. 同步更新引用
**规则**: 当模块/代码更新时，要同步相关引用了该模块/代码的地方，确保不会因为更新而导致功能异常。
**指南**:
- **修改函数签名**: 必须搜索并更新所有调用该函数的地方。
- **重命名文件**: 必须更新所有 import 路径。
- **修改组件 Props**: 必须更新所有使用该组件的地方。

**示例**:
修改了 `getUser(id)` 为 `getUser(id, options)`。
**行动**:
全局搜索 `getUser`，逐一检查并更新调用代码。
