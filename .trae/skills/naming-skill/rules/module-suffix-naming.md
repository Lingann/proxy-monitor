# 模块后缀规范 (Module Suffix Naming)

本文件定义了特定类型模块的文件名后缀规范。

## 规则列表

### 1. 特定模式后缀
**规则**: 创建/构造/生成/处理/转换/解析/校验模块的文件名必须包含明确的后缀。

**列表**:
- **Create**: `xxx-xxx-creator.ts`
- **Constructor**: `xxx-xxx-constructor.ts`
- **Generator**: `xxx-xxx-generator.ts`
- **Handler**: `xxx-xxx-handler.ts`
- **Transformer**: `xxx-xxx-transformer.ts`
- **Parser**: `xxx-xxx-parser.ts`
- **Validator**: `xxx-xxx-validator.ts`

**示例**:
- `user-data-validator.ts`
- `api-response-handler.ts`
- `report-generator.ts`
