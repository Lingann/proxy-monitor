# 风格指南 (Style Guide)

本文件定义了项目的整体视觉风格。

## 规则列表

### 1. Notion 风格
**规则**: 默认风格参考笔记应用 Notion 的配色和布局风格。
**特点**:
- **极简主义**: 大量的留白，清晰的排版。
- **柔和的配色**: 使用低饱和度的颜色，避免刺眼的对比。
- **功能导向**: 界面元素服务于内容，减少装饰性元素。
- **圆角与阴影**: 适度的圆角（通常 4px - 8px）和细腻的阴影。

**参考**:
- 字体颜色：通常不是纯黑，而是深灰（如 `#37352f`）。
- 背景色：纯白或极浅的灰色。
- 交互：快速、流畅，无过度的动画。
