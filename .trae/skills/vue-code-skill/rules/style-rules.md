# 样式规范 (Style Rules)

本文件定义了 Vue 组件的样式处理方式。

## 规则列表

### 1. 独立样式文件
**规则**: 每个组件/页面都有自己的样式文件，并且文件名与组件/页面文件名严格保持一致（使用中划线命名）。
**指南**: 推荐使用 CSS Modules 以避免全局污染，例如 `component-name.module.scss`。

### 2. 全局样式引用
**规则**: 项目中应该存在公共的样式/全局变量文件，例如：`app-styles.scss`。
**指南**: 组件样式中可以引用全局变量，但应避免在组件中定义全局样式。

### 3. CSS 变量使用
**规则**: 使用 CSS 变量定义全局的样式变量，参考 `css-style-skill` 中的规则。
